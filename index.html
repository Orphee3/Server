orphee_api

Orphee API is an api for the project Orphee : https://github.com/Orphee3

«Orphée» is a smartphone application enabling you to create and edit music in a fun and intuitive manner, alone or with friends and to share your musical creations with the user community or selected members. It is also possible, if authorized by the composer, to modify someone else’s music to add your personal touch.

  • dateCreation (Date), default: Date.now

  • creations (Array), id creation
